แนะนำ CSS ขั้นพื้นฐาน

CSS คือ ภาษาที่ใช้สำหรับตกแต่งเอกสาร HTML/XHTML ให้มีหน้าตา สีสัน ระยะห่าง พื้นหลัง เส้นขอบและอื่นๆ ตามที่ต้องการ

CSS ย่อมาจาก Cascading Style Sheets

มีลักษณะเป็นภาษาที่มีรูปแบบในการเขียน Syntax แบบเฉพาะและได้ถูกกำหนดมาตรฐานโดย W3C

เป็นภาษาหนึ่งในการตกแต่งเว็บไซต์ ได้รับความนิยมอย่างแพร่หลาย

                    body {
                        background-color: lightblue;
                    }
                    
                    h1 {
                        color: white;
                        text-align: center;
                    }
                    
                    p {
                        font-family: verdana;
                        font-size: 20px;
                    }






การเกิดขึ้นของ CSS

ในช่วงกลางทศวรรษที่ 1990 เว็บเบราเซอร์เริ่มมีการสนับสนุน HTML ในการสร้างหน้าเว็บ แต่การควบคุมการแสดงผลยังจำกัดอยู่ HTML ถูกออกแบบเพื่อกำหนดโครงสร้างของข้อมูล แต่ไม่มีเครื่องมือที่ดีในการควบคุมการแสดงผล


ในปี 1994 Håkon Wium Lie ได้เสนอแนวคิดของ CSS ขึ้นในเอกสารที่ชื่อว่า "Cascading HTML Style Sheets" ซึ่งได้รับการสนับสนุนจาก Tim Berners-Lee ผู้คิดค้น World Wide Web


ในเดือนธันวาคม 1996, World Wide Web Consortium (W3C) ได้เผยแพร่ CSS ระดับ 1 (CSS1)

เป็นมาตรฐานแรกของ CSS มาตรฐานนี้ครอบคลุมคุณสมบัติพื้นฐาน เช่น ฟอนต์ สี และการวางตำแหน่ง

ขององค์ประกอบ HTML


CSS2 ได้รับการเผยแพร่ในเดือนพฤษภาคม 1998 โดยเพิ่มคุณสมบัติใหม่ ๆ เช่น การสนับสนุนสื่อชนิด

ต่าง ๆ (media types) และการวางตำแหน่งองค์ประกอบที่ซับซ้อนยิ่งขึ้น


CSS3 เป็นการพัฒนาที่สำคัญของ CSS โดยแบ่งออกเป็นโมดูลย่อย ๆ เพื่อให้สามารถพัฒนาและยอมรับได้ง่ายขึ้น

โมดูลเหล่านี้รวมถึงการเปลี่ยนแปลง (transitions), แอนิเมชัน (animations), การวางเลเยอร์ (flexbox และ grid) และคุณสมบัติอื่น ๆ

CSS มีบทบาทสำคัญในการพัฒนาเว็บสมัยใหม่ โดยทำให้สามารถสร้างเว็บเพจที่มีการออกแบบและการใช้งานที่สวยงามได้อย่างง่ายดาย








ข้อดีของ CSS

แยกการจัดรูปแบบออกจากเนื้อหา

CSS ช่วยแยกการจัดรูปแบบและการแสดงผลออกจากเนื้อหา HTML ทำให้การจัดการเนื้อหาและการออกแบบเป็นไปอย่างมีประสิทธิภาพมากขึ้น

ประหยัดเวลาและลดความซ้ำซ้อน

การใช้ CSS ช่วยให้สามารถนำสไตล์ไปใช้ซ้ำในหลายๆ หน้าเพียงแค่เขียนโค้ดครั้งเดียว ทำให้ลดเวลาที่ใช้ในการเขียนโค้ด

การควบคุมรูปแบบที่ดีกว่า

CSS ช่วยให้สามารถควบคุมการแสดงผลได้มากขึ้น เช่น การตั้งค่าการจัดวาง (layout), สี, ฟอนต์ และการตอบสนอง (responsive design)

ปรับปรุงประสิทธิภาพเว็บไซต์

การใช้ CSS ช่วยลดขนาดของไฟล์ HTML เนื่องจากไม่ต้องใส่โค้ดการจัดรูปแบบในแต่ละหน้า ทำให้เว็บไซต์โหลดเร็วขึ้น

การสนับสนุนการออกแบบที่ตอบสนอง

CSS ช่วยให้การออกแบบเว็บไซต์ที่ตอบสนองต่ออุปกรณ์ต่างๆ เช่น มือถือ แท็บเล็ต และเดสก์ท็อป เป็นไปอย่างง่ายดาย

การบำรุงรักษาที่ง่าย

การเปลี่ยนแปลงรูปแบบสามารถทำได้ง่ายและรวดเร็ว เพียงแค่ปรับแต่งไฟล์ CSS เดียวเท่านั้น

ข้อเสียของ CSS

ปัญหาการเข้ากันได้

บางเบราว์เซอร์อาจไม่สนับสนุนคุณสมบัติ CSS ทั้งหมด ทำให้เกิดปัญหาการแสดงผลไม่เหมือนกันในเบราว์เซอร์ต่างๆ

ความซับซ้อนในการจัดการโค้ด

ในโครงการใหญ่ๆ การจัดการ CSS อาจซับซ้อนและยุ่งยากถ้าไม่มีการวางแผนและโครงสร้างที่ดี

การเรียนรู้และใช้งาน

สำหรับผู้เริ่มต้น การเรียนรู้ CSS อาจจะต้องใช้เวลาและความพยายามในการเข้าใจแนวคิดและวิธีการใช้งาน

ขาดการแยกเฉพาะหน้า

การใช้ CSS ในการจัดรูปแบบอาจส่งผลกระทบต่อองค์ประกอบอื่นๆ ในหน้าเว็บเดียวกัน ทำให้ต้องระวังในการเขียนโค้ด

การแก้ไขปัญหาการทับซ้อน

CSS ใช้หลักการการทับซ้อน (cascade) และความเฉพาะเจาะจง (specificity) ซึ่งอาจทำให้การจัดการและแก้ไขปัญหาในการแสดงผลเป็นเรื่องที่ซับซ้อน







หลักการเขียน

ตัวเลือกชี้ไปยังองค์ประกอบ HTML ที่คุณต้องการจัดรูปแบบ
บล็อกการประกาศประกอบด้วยการประกาศอย่างน้อยหนึ่งรายการที่คั่นด้วย เครื่องหมายอัฒภาค
การประกาศแต่ละรายการประกอบด้วยชื่อคุณสมบัติ CSS และค่า คั่นด้วยเครื่องหมายทวิภาค
การประกาศ CSS หลายรายการจะถูกคั่นด้วยเครื่องหมายอัฒภาคและการประกาศ บล็อกล้อมรอบด้วยวงเล็บปีกกาหยิก

ตัวอย่าง CSS


                                        .CTEXT {
                                            color: black;

                                            font-family: "Kanit", sans-serif;

                                            text-align: left;

                                            font-size: 26px;






หลักการเรียกใช้งาน

เมื่อเบราว์เซอร์อ่านสไตล์ชีต มันจะจัดรูปแบบเอกสาร HTML ตาม ข้อมูลในสไตล์ชีต

3 วิธีการเพิ่ม CSS ในเว็บไซต์

External CSS

ด้วย สไตล์ชีตภายนอกคุณสามารถเปลี่ยนรูปลักษณ์ของทั้งเว็บไซต์ได้โดยการเปลี่ยนแปลง ไฟล์เดียวเท่านั้น! แต่ละหน้า HTML ต้องมีการอ้างอิงไปยังแฟ้มสไตล์ชีตภายนอกภายใน องค์ประกอบ ภายในส่วนหัว


External


    <html>
        <head>
            <title>HTML Tutorial</title>
            <link rel="stylesheet" href="Contentstyle.css"> ใช้บรรทัดนี้ในการเรียกใช้ CSS ภายนอก
        </head>
    <body>

        <h1>This is a heading</h1>
        <p>This is a paragraph.</p>

    </body>
</html>

Internal CSS

อาจใช้สไตล์ชีตภายในหากหน้า HTML หน้าเดียวมีสไตล์ที่ไม่ซ้ํากัน ลักษณะภายในถูกกําหนดภายในองค์ประกอบ สไตล์ ภายในส่วนหัว ตอน

Internal


    <html>
        <head>
            <title>HTML Tutorial</title>
        </head>
        
            .box {
                background-color: #f8f8f8;
                border-left: 5px solid #4CAF50;
                padding: 16px;
                font-family: Consolas, 'Courier New', monospace;
                margin: 20px 0;
                box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
                border-radius: 4px;
            }
            </style>
    <body>

        <h1>This is a heading</h1>
        <p class="box">This is a paragraph.</p>

    </body>
</html>

Inline CSS

ลักษณะแบบอินไลน์อาจใช้เพื่อใช้ลักษณะที่ไม่ซ้ํากันสําหรับองค์ประกอบเดียว หากต้องการใช้สไตล์อินไลน์ ให้เพิ่มแอตทริบิวต์ style ให้กับองค์ประกอบที่เกี่ยวข้อง พื้นที่ style สามารถมีคุณสมบัติ CSS ใดก็ได้

Inline


<html>
    <head>
        <title>HTML Tutorial</title>
        <link rel="stylesheet" href="Contentstyle.css">
    </head>
<body>

    <h1 style="color:blue;text-align:center;">This is a heading</h1>
    <p style="color:red;">This is a paragraph.</p>

</body>
</html>






การใช้งาน Font

การเลือกแบบอักษรเป็นสิ่งสําคัญ


การเลือกแบบอักษรที่เหมาะสมมีผลกระทบอย่างมากต่อประสบการณ์ของผู้อ่านเว็บไซต์.


แบบอักษรที่เหมาะสมสามารถสร้างเอกลักษณ์ให้กับเว็บไซต์ของคุณได้


การใช้แบบอักษรที่อ่านง่ายเป็นสิ่งสําคัญ แบบอักษรเพิ่ม ค่าของข้อความของคุณ สิ่งสําคัญคือต้องเลือกสีและขนาดตัวอักษรที่ถูกต้อง สําหรับแบบอักษร


ตระกูลฟอนต์ทั่วไป


ใน CSS มีตระกูลฟอนต์ทั่วไปห้าตระกูล:


1.แบบอักษร Serif มีจังหวะเล็กๆ ที่ขอบของตัวอักษรแต่ละตัว พวกเขาสร้างความรู้สึกเป็นทางการและความสง่างาม


2.แบบอักษร Sans-serif มีเส้นที่สะอาดตา (ไม่มีจังหวะเล็ก ๆ ติดอยู่) พวกเขาสร้างรูปลักษณ์ที่ทันสมัยและเรียบง่าย


3.แบบอักษร Monospace - ที่นี่ตัวอักษรทั้งหมดมีความกว้างคงที่เท่ากัน พวกเขาสร้างรูปลักษณ์เชิงกล


4.แบบอักษรตัวเขียนเลียนแบบลายมือของมนุษย์


5.แบบอักษรแฟนตาซีเป็นแบบอักษรตกแต่งเล่นได้


รูปแบบตัวอักษร


คุณสมบัติส่วนใหญ่จะใช้เพื่อระบุข้อความตัวเอียง font-style


คุณสมบัตินี้มีค่าสามค่า:

normal - ข้อความแสดงตามปกติ

ตัวเอียง - ข้อความแสดงเป็นตัวเอียง

เฉียง - ข้อความ "เอน" (เฉียงคล้ายกับตัวเอียงมาก แต่รองรับน้อยกว่า)


ตัวแปรแบบอักษร

คุณสมบัติระบุว่าข้อความควร แสดงเป็นแบบอักษรตัวพิมพ์ใหญ่ขนาดเล็กfont-variant

ในแบบอักษรตัวพิมพ์ใหญ่ ตัวพิมพ์เล็กทั้งหมดจะถูกแปลงเป็นตัวพิมพ์ใหญ่ ตัวอักษร อย่างไรก็ตาม

ตัวอักษรตัวพิมพ์ใหญ่ที่แปลงแล้วจะปรากฏในขนาดตัวอักษรที่เล็กลง กว่าตัวอักษรตัวพิมพ์ใหญ่ดั้งเดิมในข้อความ



ขนาดตัวอักษร

คุณสมบัติจะกําหนดขนาดของข้อความ font-size


ความสามารถในการจัดการขนาดข้อความเป็นสิ่งสําคัญในการออกแบบเว็บ อย่างไรก็ตาม

คุณไม่ควรใช้การปรับขนาดตัวอักษรเพื่อทําให้ย่อหน้าดูเหมือนหัวเรื่อง หรือ หัวเรื่องมีลักษณะเหมือนย่อหน้า


ใช้แท็ก HTML ที่เหมาะสมเสมอ เช่น <h1> - <h6> สําหรับหัวเรื่อง และ <p> สําหรับ ย่อหน้า


ค่า font-size สามารถเป็น ขนาดสัมบูรณ์หรือสัมพัทธ์


ขนาดสัมบูรณ์:


ตั้งค่าข้อความเป็นขนาดที่ระบุ

ไม่อนุญาตให้ผู้ใช้เปลี่ยนขนาดข้อความในทุกเบราว์เซอร์ (ไม่ดีด้วยเหตุผลด้านการเข้าถึง)

ขนาดสัมบูรณ์มีประโยชน์เมื่อทราบขนาดทางกายภาพของเอาต์พุต


ขนาดสัมพัทธ์:


ตั้งค่าขนาดที่สัมพันธ์กับองค์ประกอบโดยรอบ <>อนุญาตให้ผู้ใช้เปลี่ยนขนาดข้อความในเบราว์เซอร์

ระบุสีโดยใช้ชื่อสีที่กำหนดไว้ล่วงหน้า หรือค่า RGB, HEX, HSL, RGBA, HSLA


ตัวอย่างชื่อสีที่กำหนดไว้ล่วงหน้า

Red
Blue
green
Yellow

Color


                                    p {
                                        Color: red;
                                    }

                                    h1 {
                                        Color: red;
                                    }

ค่า RGB


ใน CSS สามารถระบุสีเป็นค่า RGB ได้โดยใช้สูตรนี้:

rgb( แดง, เขียว , น้ำเงิน )

พารามิเตอร์แต่ละตัว (สีแดง สีเขียว และสีน้ำเงิน) จะกำหนดความเข้มของสีระหว่าง 0 ถึง 255

ตัวอย่างเช่น rgb(255, 0, 0) จะแสดงเป็นสีแดง เนื่องจากสีแดงถูกตั้งค่าเป็นค่าสูงสุด (255) และค่าอื่นๆ จะถูกตั้งค่าเป็น 0

หากต้องการแสดงสีดำ ให้ตั้งค่าพารามิเตอร์สีทั้งหมดเป็น 0 เช่นนี้: rgb(0, 0, 0)

หากต้องการแสดงสีขาว ให้ตั้งค่าพารามิเตอร์สีทั้งหมดเป็น 255 เช่นนี้: rgb(255, 255, 255)



ค่า HEX


ใน CSS สามารถระบุสีได้โดยใช้ค่าเลขฐานสิบหกในรูปแบบ:

# rrggbb

โดยที่ rr (สีแดง), gg (สีเขียว) และ bb (สีน้ำเงิน) เป็นค่าเลขฐานสิบหกระหว่าง 00 ถึง ff (เช่นเดียวกับเลขฐานสิบ 0-255)

ตัวอย่างเช่น #ff0000 จะแสดงเป็นสีแดง เนื่องจากสีแดงถูกตั้งค่าเป็นค่าสูงสุด (ff) และสีอื่นๆ จะถูกตั้งค่าเป็นค่าต่ำสุด (00)

หากต้องการแสดงเป็นสีดำ ให้ตั้งค่าทั้งหมดเป็น 00 เช่นนี้: #000000

หากต้องการแสดงเป็นสีขาว ให้ตั้งค่าทั้งหมดเป็น ff เช่นนี้: #ffffff


ค่า HSL


ใน CSS สามารถระบุสีได้โดยใช้เฉดสี ความอิ่มตัว และความสว่าง (HSL) ในรูปแบบ:

hsl( เฉดสี , ​​ความอิ่มตัว , ความสว่าง )

ฮิว (Hue) คือระดับบนวงล้อสีตั้งแต่ 0 ถึง 360 โดย 0 คือสีแดง 120 คือสีเขียว และ 240 คือสีน้ำเงิน

ความอิ่มตัวเป็นค่าเปอร์เซ็นต์ โดย 0% หมายถึงเฉดสีเทา และ 100% คือสีเต็มรูปแบบ

ความสว่างก็เป็นเปอร์เซ็นต์เช่นกัน 0% คือสีดำ 50% คือไม่สว่างหรือมืด 100% คือสีขาว





การใช้งาน background

คุณสมบัติพื้นหลัง CSS ใช้เพื่อเพิ่มเอฟเฟ็กต์พื้นหลังให้กับองค์ประกอบ

CSS background-color

คุณสมบัติ background-color จะเอาไว้ใช้ระบุสีพื้นหลังขององค์ประกอบ

ตัวอย่าง CSS


                                        
body {
    background-color: #f0f0f0;
}

h1 {
    background-color: red;
}






การใช้งาน Comment

การใช้งาน Comment ใน CSS เป็นวิธีหนึ่งในการใส่ข้อความอธิบายหรือบันทึกหมายเหตุภายในไฟล์ CSS ซึ่งช่วยให้โค้ดมีความชัดเจนและง่ายต่อการเข้าใจ โดยเฉพาะเมื่อมีการทำงานร่วมกับผู้อื่น หรือเมื่อกลับมาแก้ไขโค้ดในภายหลัง

วิธีการเขียน Comment ใน CSS


/* ข้อความที่ต้องการใส่เป็น Comment */

ตัวอย่าง CSS


                                        /* ตั้งค่าพื้นหลังของหน้าเว็บ */
body {
    background-color: #f0f0f0;
}

/* ตั้งค่าสีของข้อความ */
p {
    color: #333;
}

/* กำหนดขนาดของฟอนต์ */
h1 {
    font-size: 2em;
}

ในตัวอย่างข้างต้น ข้อความที่อยู่ภายใน /* ... */ จะไม่ถูกประมวลผลโดยเบราว์เซอร์ ทำให้สามารถใส่คำอธิบายหรือหมายเหตุในโค้ด CSS ได้ตามต้องการ


ประโยชน์ของการใช้ Comment ใน CSS


1.เพิ่มความเข้าใจ: ทำให้ผู้อ่านเข้าใจโค้ดได้ง่ายขึ้น โดยเฉพาะในกรณีที่โค้ดมีความซับซ้อน

2.บันทึกหมายเหตุ: ช่วยบันทึกความคิดหรือเหตุผลที่ใช้ในการเขียนโค้ด

3.ปิดการทำงานชั่วคราว: สามารถใช้ Comment เพื่อปิดการทำงานของบางส่วนของโค้ดชั่วคราวในระหว่างการทดสอบ







การใช้งาน Border

border ใน CSS ใช้สำหรับกำหนดขอบรอบองค์ประกอบ (elements) บนหน้าเว็บ
การใช้งาน border มีความยืดหยุ่นสูง สามารถกำหนดสี ความหนา และรูปแบบของเส้นขอบได้ตามต้องการ


การใช้งานพื้นฐานของ border

คุณสามารถกำหนดขอบให้กับองค์ประกอบ HTML โดยใช้คุณสมบัติ border ดังนี้

ตัวอย่าง CSS


                                        selector {
                                            border: [ความหนา] [รูปแบบเส้น] [สี];
                                        }

ตัวอย่างการใช้งาน

ตัวอย่าง CSS


                                                .Start {
                                                    background-color: rgb(0, 0, 0);
                                                    border: 1px solid transparent;
                                                    text-align: center;
                                                    color: #ffffff;
                                                    margin-top: 50px;
                                                    margin-bottom: 50px;
                                                    border-radius: 11px;
                                                    padding: 30px 30px 30px 30px;
                                                    margin-right: 10vh;
                                                    justify-content: space-between;
                                                    text-decoration: none;
                                                    font-family: "Kanit", sans-serif;
                                                }

คุณสมบัติของ border

คุณสามารถกำหนดคุณสมบัติของ border แยกกันได้ ดังนี้

1. border-width: กำหนดความหนาของเส้นขอบ เช่น border-width: 2px
2. border-style: กำหนดรูปแบบของเส้นขอบ เช่น solid, dashed, dotted, double, groove, ridge, inset, outset, หรือ none (ไม่มีเส้นขอบ)
3. border-color: กำหนดสีของเส้นขอบ เช่น border-color: green


การกำหนดขอบเฉพาะด้าน

border-top: กำหนดขอบด้านบน

border-right: กำหนดขอบด้านขวา

border-bottom: กำหนดขอบด้านล่าง

border-left: กำหนดขอบด้านซ้าย

                    .Quizbutton {

                            border: 1px solid transparent;
                            text-align: center;
                            color: #ffffff;
                            padding-top: 15px;
                            border-radius: 11px;
                            width: 120px;
                             height: 80px;
                            text-decoration: none;
                            font-family: "Kanit", sans-serif;
                        }






การใช้งาน Margins

margin ใน CSS ใช้สำหรับกำหนดระยะห่างรอบนอกขององค์ประกอบ (elements) เพื่อเว้นระยะห่างระหว่างองค์ประกอบนั้นกับองค์ประกอบอื่น ๆ บนหน้าเว็บ โดย margin สามารถกำหนดได้ทั้งแบบรวมและแบบแยกแต่ละด้าน


การกำหนด margin

การกำหนด margin เฉพาะด้านต่าง ๆ ได้ดังนี้


- margin-top: กำหนดระยะห่างด้านบน

- margin-right: กำหนดระยะห่างด้านขวา

- margin-bottom: กำหนดระยะห่างด้านล่าง

- margin-left: กำหนดระยะห่างด้านซ้าย


ค่าของ margin


- หน่วยพิกเซล (px): เป็นหน่วยที่ใช้งานมากที่สุด

- เปอร์เซ็นต์ (%): ใช้เป็นเปอร์เซ็นต์ของขนาดขององค์ประกอบแม่

- หน่วย em: อ้างอิงจากขนาดฟอนต์ขององค์ประกอบ

- auto: ใช้สำหรับการจัดวางองค์ประกอบ

- :0: ใช้เมื่อต้องการไม่มีระยะห่าง

ตัวอย่าง CSS


                                        .ContentButtonGroup {
                                            /*background-color: rgb(87, 87, 150);*/
                                            display: flex;
                                            flex-direction: row;
                                            margin-top: 15px;
                                        }

การกำหนด Padding

- padding-top: กำหนดระยะห่างด้านบน

- padding-bottom: กำหนดระยะห่างด้านล่าง

ค่าของ padding

- เปอร์เซ็นต์ (%): ใช้เป็นเปอร์เซ็นต์ของขนาดขององค์ประกอบแม่

- หน่วย em: อ้างอิงจากขนาดฟอนต์ขององค์ประกอบ

- auto: ใช้สำหรับการจัดวางองค์ประกอบ

- :0: ใช้เมื่อต้องการไม่มีระยะห่าง

                                        h2 {
                                            color: #2c3e50;
                                            font-size: 48px;
                                            padding-left: 20px;
                                        }






การใช้งาน Height และ Width

width ใน CSS ใช้สำหรับกำหนดความกว้างขององค์ประกอบ (elements) บนหน้าเว็บ สามารถกำหนดความกว้างเป็นค่าคงที่หรือแบบยืดหยุ่นตามขนาดของหน้าจอหรือ container ที่อยู่รอบ ๆ องค์ประกอบนั้น


การใช้งานพื้นฐานของ width


ตัวอย่าง การใช้งาน


                                        selector {
                                            width: [ค่า];
                                        }

ค่าของ height , width


- พิกเซล (px): หน่วยมาตรฐานที่ใช้มากที่สุด เช่น width: 300px

- เปอร์เซ็นต์ (%): กำหนดความกว้างเป็นเปอร์เซ็นต์ของ container ที่อยู่รอบ ๆ เช่น width: 50%

- หน่วย em: กำหนดความกว้างเป็นหน่วยที่อิงตามขนาดฟอนต์ขององค์ประกอบ เช่น width: 10em

- หน่วย rem: อิงจากขนาดฟอนต์ของ root element (html) เช่น width: 20rem

- หน่วยอื่น ๆ: เช่น vw (viewport width) และ vh (viewport height) ที่ใช้กำหนดความกว้างตามขนาดของหน้าจอ เช่น width: 50vw;


ตัวอย่าง CSS


                                                .box {
                                                    width: 20px;
                                                    height: 20px;
                                                    background-color: blue;
                                                }
